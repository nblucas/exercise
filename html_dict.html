Descrição das TAGs HTML utilizadas

Retornar à página pessoal.
  • !DOCTYPE: Definir a versão do HTML utilizada no documento;
  • html: Envelopa head e body;
  • head: Envelopa as tags que descrevem e configuram a aplicação;
  • meta: Define metadados da aplicação;
  • title: Define o título na aba do navegador;
  • link: Faz conexões entre arquivos entre arquivos de iamgem, texto, folha de estilos..;
  • body: Envelopa as tags que descrevem a estrutura da aplicação e seu conteúdo;
  • header: Tag divisória semântica para definição de cabeçalho;
  • nav: Tag divisória semântica para definição de navegação;
  • ul: Define uma lista não ordenada;
  • li: Define um item de uma lista;
  • a: Define uma "âncora" para acessar outras seções e páginas web;
  • main: Tag divisória semântica para definição do conteúdo principal da aplicação;
  • section: Tag divisória semântica para definição de uma nova seção;
  • aside: Tag divisória semântica para definição de uma nova seção lateral;
  • div: Tag divisória, ou seja, define uma nova seção (não é semântica);
  • p: Define um novo parágrafo;
  • article: Tag divisória semântica para definição de artigo;
  • h1: Tag de texto semântica que define um título importante;
  • h2: Tag de texto semântica que define um título importante, mas não tanto quanto o h1;
  • h3: Tag de texto semântica que define um título importante, mas não tanto quanto o h2;
  • footer: Tag divisória semântica para definição de rodapé;
  • address: Tag semântica para envelopar dados sobre endereço;
  • style: Tag utilizada para escrever CSS diretamente no documento .html;
